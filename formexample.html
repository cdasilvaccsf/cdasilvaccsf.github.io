HTML5 and Forms

1. The autofocus attribute specifies that a field should automatically get focus when a page is loaded. Note: The autofocus attribute works with all input types.



2. The email type is used for input fields that should contain an e-mail address. The value of the email field is automatically validated when the form is submitted. It also helps the user when filling out the form as the browser will present "suggestions" of previous email addresses the user typed.



3. The url type is used for input fields that should contain a URL address. The value of the url field is automatically validated when the form is submitted but notice that the user will need to type the COMPLETE URL including http or https.

FINAL OBSERVATION

All the examples shown above include the label tag that needs to be used in forms to provide accessibility! There are two ways to use the label tag:
<label for="yourage">Your age: </label><input type="number" id="yourage" name="age" min="18">
where the value of the for attribute is the value of the id attribute of the tag that the label refers to

OR

<label>Your age: <input type="number" id="yourage" name="age" min="18"></label>
where the label tag is surrounding the text/content of the label and the input tag the label refers to


It is important to include the name attribute in the field areas of your form (input, textarea, etc.). It identifies the element in a JavaScript or server-side programming when the form is submitted. Only form elements with a name attribute will have their values passed when submitting a form! So, get used to code the name attribute in your forms!
